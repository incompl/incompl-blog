--- layout: post title: Concise JavaScript Image Preloading published: true meta: Short Link: "972" _edit_last: "1" _wp_old_slug: "" tags: - programming type: post status: publish --- For some reason, all over the Internet people preload images like this: {% highlight javascript %}var img = new Image(); img.src = "image.jpg";{% endhighlight %} There is no reason for this code to be two lines. I preload my images like this: {% highlight javascript %}new Image().src = "image.jpg";{% endhighlight %} It would be hard to come up with a more minor nitpick than this, but if we're going to write this code hundreds of times during our lives, why not keep it simple?
